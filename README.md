# gustaferiksson/homebrew-tap

A [Homebrew](https://brew.sh) tap for my macOS apps.

## Amped

A tiny native menu bar app that keeps your Mac awake — even with the lid closed.
([source](https://github.com/gustaferiksson/amped))

```sh
brew install --cask gustaferiksson/tap/amped
```

The cask is updated automatically by the release workflow in the Amped repo on
every tagged release.
